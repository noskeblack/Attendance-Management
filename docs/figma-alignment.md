# 画面設計 DG01（Figma・素材）との関係

## 要件

- **DG01**: 参考デザイン（Figma）に準拠すること、COACHTECH 提供素材の利用。

## 本リポジトリの方針

- **Figma ファイル・画像素材はリポジトリに同梱していません**（受講生向け配布物のため、各自の入手状況に依存します）。
- 画面は Blade + `resources/css/app.css` の **`ct-*` ユーティリティ**でレイアウト・コンポーネントを揃えています。
- **ピクセル完全一致**は、Figma と素材（フォント・アイコン・画像）をローカルに配置したうえで、個別に微調整してください。

## 採点・提出でやると良いこと

1. 教材から **Figma の参照リンク**と **素材パッケージ**を取得する。  
2. 必要なら `public/` 配下に画像を配置し、Blade で参照する。  
3. **DG02（1400–1540px）** はブラウザで主要画面を確認し、`app.css` のメディアクエリを調整する。
