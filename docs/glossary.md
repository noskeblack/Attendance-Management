# 用語・表記（要件シートとの整合）

## 勤務中 / 出勤中

- **本アプリの画面・ステータス表示の正規表記は「出勤中」**です（`AttendanceStatus::Working`）。
- 要件シート「テストケース一覧」の **出勤機能（6.0）** における期待挙動の **「勤務中」** は、**同一状態（勤務開始後〜退勤前かつ休憩でない）**を指す表現として、**「出勤中」と同義**とみなして実装・テストしています。
- シート内では **ステータス確認（5.0）** 側が **「出勤中」** 表記になっており、**6.0 だけ「勤務中」**となっているため、**表記ゆれ（誤記）**の可能性が高いです。採点・レビュー時は **画面設計タブ**の表記を優先してください。

## 勤怠修正バリデーション（FN029）

- **出勤・退勤の前後関係が不適切**な場合のメッセージは、機能要件 **FN029** に従い **「出勤時間もしくは退勤時間が不適切な値です」** としています（`clock_in` フィールドに付与）。
- **休憩開始**が出勤より前、または退勤より後の場合は **「休憩時間が不適切な値です」**、**休憩終了**が退勤より後の場合は **「休憩時間もしくは退勤時間が不適切な値です」** です。

## 勤怠登録画面 / 勤怠打刻画面

- メール認証完了後の遷移先は **`/attendance`（勤怠打刻）** です。シート上の「勤怠**登録**」という名称と **画面タイトル「勤怠打刻」** が異なる場合がありますが、**同一画面（打刻・ステータス確認のホーム）**を指します。
