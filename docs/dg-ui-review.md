# DG01/DG02 画面チェック（2026-03-21）

要件シートの `DG01`（Figma 準拠）と `DG02`（1400-1540px のレイアウト崩れ防止）に対して、
主要画面の Blade と共通 CSS を確認し、差分を反映した。

## 判定基準

- **DG01**: `ct-*` コンポーネント体系（カード・フォーム・ボタン・テーブル）に統一されていること
- **DG02**: 固定幅レイアウト/旧スタイルを避け、1400-1540px で折返し・横スクロールが破綻しないこと

| PG02 | 一般ログイン | 対応済み | 余白指定を共通クラス化 |
| PG03 | 勤怠打刻 | 対応済み | アクション余白を共通クラス化 |

## 追加した共通スタイル

- `ct-actions--mt`
- `ct-actions--mb`
- `ct-row`
- `ct-row--mb`
- `ct-block-mt`
- `ct-subtitle--spaced`

これにより画面間の余白・ボタン・入力行の振る舞いを統一し、DG01/DG02 の確認観点をコード上で満たしやすくした。
