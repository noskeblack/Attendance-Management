# Blade/CSS 画面チェック（2026-03-21）

## 判定基準

- **コンポーネント統一**: `ct-*` 体系（カード・フォーム・ボタン・テーブル）に揃っていること
- **中幅ビューポート**: 固定幅レイアウトや旧スタイルを避け、1400–1540px で折返し・横スクロールが破綻しないこと

| 一般ログイン | 対応済み | 余白指定を共通クラス化 |

## 追加した共通スタイル

- `ct-actions--mt`
- `ct-actions--mb`
- `ct-row`
- `ct-row--mb`
- `ct-block-mt`
- `ct-subtitle--spaced`

これにより画面間の余白・ボタン・入力行の振る舞いを統一し、上記の確認観点をコード上で満たしやすくした。
